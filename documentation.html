Getting Started

Welcome to Weaveries — a writing and note-taking plugin for WordPress. Use it to create interconnected notes, plan stories, build world-bibles, and more.

After activating the plugin you will find a new Weaves menu in your WordPress admin sidebar.

Weave Types

Types let you categorise your Weaves — for example Person, Place, Story, or Task. Each type can have its own colour, icon, custom fields, and status fields.

  1. Go to Weaves → Settings → Weave Types
  2. Enter a name and click Add Type
  3. Use Edit Appearance to set a colour and icon
  4. Add custom fields and assign status fields below each type

Custom Fields

Custom fields let you store structured information on a Weave — such as a birth date, a URL, or a dropdown value.

Supported field types: Text, Text Area, Number, Date, URL, Email, Dropdown, Checkbox.

Fields are defined per Weave Type and appear in the Custom Fields metabox when editing a Weave of that type.

Statuses

Status fields let you track the state of a Weave — for example a Task Status with options To Do, In Progress, and Done.

  1. Go to Weaves → Settings → Statuses
  2. Create a status field (e.g. Task Status)
  3. Add options, each with a colour and a meaning (New, In Progress, or Complete)
  4. Assign the status field to one or more Weave Types in the Weave Types tab

The main status field for a type is shown prominently on the Weave edit screen.

Relationships

You can link any two Weaves together using the Relationships metabox on the Weave edit screen. Optionally assign a relationship type (e.g. References, Works At) to give the link meaning.

Relationship types are managed under Weaves → Settings → Relationship Types.

Shortcodes

Use the [weave] shortcode to embed Weave data anywhere on your site.

ShortcodeOutput
[weave field="field_name"]The value of a custom field
[weave field="title"]The Weave title
[weave field="content"]The Weave body content
[weave status="status-slug"]A colour-coded status badge
[weave field="field_name" id="42"]Field from a specific Weave

Click any shortcode chip in the Settings page or Custom Fields metabox to copy it to your clipboard.

Frontend Pages & Permalinks

Each Weave has its own public page on the frontend. The URL prefix defaults to /wiki/, giving URLs like yourdomain.com/wiki/my-weave.

You can change the prefix under Weaves → Settings → General. Note that changing it will break existing URLs, so set it before your site goes live.

General Tips

  • Assign multiple Weave Types to a single Weave to combine their fields and statuses
  • Use relationship types to build a meaningful knowledge graph
  • The default status field is pre-selected when no type is assigned
